Colours

In CSS colours can be specified using different types of values, such as color names, RGB, HEX, exc.

Firstly, HTML supports 140 standard colour names: red, green, white, black, yellow, orange, gray, violet, MediumSeaGreen, DodgerBrue, etc.

Moreover, you can specify colour using intensities of red, green and blue colours by formula:

rgb(red,green,blue)

The intencity of these colours can be defined between 0 and 255.

HEX value is also often used way of specifying a colour. It has the form #rrggbb where rr, gg and bb are numbers between 0 and 255 but in hexadecimal (00-ff).
