HTML Attributes

To specify additional properties and/or behaviour for the HTML element use attributes.

  • Attributes are always declared in the opening tag.
  • Self-closing attributes tags can also contain attributes.
  • Attributes come as name/value pairs: name = “value”.

Links and the href attribute

To create a link to an external web page use the <a> tag.

The link’s destination address is additional information which is specified using href attribute.

href = “URL address”


Use a full web address (an absolute link) to go to a different web page.

To access any point on the same web page use a local link which is specified with a relative URL (no need to add "http://www. ...").

Images and the scr, alt, width and height attributes

You can define HTML images with the <img> tag.

Additional properties for the image are:

  • the image source file ( src attribute )
  • size ( width and height attributes specify the width and height of the image in pixels)
  • alternative text when an image cannot be displayed ( alt attribute )

The title, style, class and id attributes

The following attributes can be applied to any tag in <body> . However, most of them are used for CSS styling and, therefore, will be examined in more detail later:

  • title – creates a tooltip for the text specified in value of this attribute and will be displayed when the mouse is over the element.

  • style - attribute provides inline CSS style rules to specify the styling of an element (color, size, font)
