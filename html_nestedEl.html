Nested elements in HTML

HTML elements can contain other elements which is called nesting.

Example:

A skeletal structure of a web page observed in the previous section is an example of nesting because <head> and <body> elements are contained within the <html> element.

The <body> element which defines the document body also contains other HTML elements (a heading <h1> and a paragraph <p>).

The content of the <html> tag are elements <head> and <body>

The content of the <body> element are elements <h1> and <p>
