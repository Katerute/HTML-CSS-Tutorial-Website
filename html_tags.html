Tags

HTML tags specify the type of an element and construct the content of a web page.

The template for a tag

The template for a tag: < tagName>content</tagName>

a tag name surrounded by angle brackets

Tags normally come in pairs: start/opening tag and end/closing tag.
But some of tags are not paired because they never enclose any content (see later).

There is a variety of tags to label different pieces of web content.

Headings

Headings are the most important structural components of the page.

A heading can be set to six levels, so are defined with tags which have the following template:

  • <h + number>
  • <h1></h1>
  • <h2></h2>
  • <h3></h3>
  • <h4></h4>
  • <h5></h5>
  • <h6></h6>
  • <h1> - the most important heading
  • <h6> - the least important heading
  • There can be only one <h1> per page. Headings of other levels can appear more than once.
  • Levels cannot be skipped:
    • <h1> → <h2> → <h3> OK
    • <h1> → <h2> → <h3> → <h2> OK
    • <h1> → <h3> X
  • Use <h1>-<h6> for headings ONLY! Do not use them to change the size of the text or make it bold.

Paragraphs

Paragraphs are commonly used as basic containers to represent text in the body section. They are defined using <p></p> tags.

It is important to know that you cannot change the output adding extra spaces or lines because the browser will remove all of those when displaying the page.

Empty HTML elements

HTML element without any content are called empty elements and do not require a closing tag.

For example, <br/> inserts a line break.
<hr/> is used to separate the content, defines a thematic break.

As you can see such tags are “closed” in the opening tag <br/>.
However, if you work with HTML5 it is not obligatory to ”close” the empty elements, they can be left as <br>.

So, if you want a new line without starting a new paragraph use the <br/> tag.

There are 2 types of lists in HTML: ordered and unordered.

A bullet point list is defined with the <ul> and ordered with the <ol> tag. Both of them are followed by <li> tags for list items.
