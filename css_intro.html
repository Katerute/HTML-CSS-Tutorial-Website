CSS Introduction

What is CSS?

  • CSS (Cascading Style Sheets) describes the appearance of HTML document.
  • In CSS you can specify how HTML elements should be displayed.
  • CSS is used to separate style from structure and behavior.
  • One CSS document can manage the layout of multiple web pages.

How CSS can be added to HTML?

There are 3 variants how to add CSS to HTML:

  • Inline – use style attribute with CSS rules as value in HTML document (bad practice because style and structure should be separated).
  • Internal – defined by <style> tag in Head section and used to control only one single HTML page.
  • External –CSS code is stored in a separate file and linked to HTML document. A link to style sheets is specified in the Head section of HTML document. It used to define a layout of multiple HTML pages.
