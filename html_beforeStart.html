Before you get started

Where to begin?

  • All web documents use HTML (Hypertext Mark-up Language) to structure and display the content.
  • Other technologies (such as CSS and Java Script) are used for the appearance of the web page.

You need to study HTML first if you want to learn web site development.

What do you need to start learning HTML?

  • Text editor → preferably Notepad++
  • A web browser → preferably Google Chrome
  • This web page in your 'Favourites' to guide you towards independent web page development ☺

How to create a HTML document?

You will always do the same operations in order to create a web page:

Open text editor → Write some HTML → Save the document in either .htm or .html file extension → Open the file in preferable browser

If you are ready open a ‘New file’ in Notepad++ and let’s get started!
