HTML Tables

To create a HTML table you have to define:

  • a table element itself with the <table> tag
  • a table header with the <th> tag (optional)
  • each table row with the <tr> tag (as many as you need)
  • each cell in the raw with the <td> tag (as many as you need)

NOTE

  1. The content of the table heading element is bold and centered by default!
  2. The table and the cells do not have borders by default!
  3. The cell elements are only containers for data. These data containers may contain all kinds of HTML elements such as text, images, lists and even other tables!

Example:

To change the size of the table, align the text, add borders, etc. you can either

  • use the following table and cell attributes:
    • border – border visibility and thickness ( = 0 – no gridlines, = 1 - gridlines, > 1 – outside border only)
    • width - width of table (specify the number of pixels or % for relative width)
    • cellspacing – distance between cells (specify the number of pixels)
    • cellpadding – distance between cell content and cell edges (specify the number of pixels)
    • colspan – merge column cells (specify the number of cells to be merged)
    • rowspan – merge row cells (specify the number of cells to be merged)

  • use style attribute (for limited number of properties)
  • use CSS for markup (the best solution, but see later)
