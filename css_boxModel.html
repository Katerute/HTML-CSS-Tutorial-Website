The CSS Box Model

Around every HTML element there are a wrapping box which consists of margin, border, padding and the content.

Border

Border is the outline of an element’s box.
It goes around content and padding and has the following properties:

  • Style → dashed/dotted/solid/double
  • Width → specify the width of the border
  • Colour → specify colour value

NOTE
border-width and border-color do not have any effect if border-style is not set

Padding

Padding creates space around the content of an element.
You can set padding for all 4 sides of an element separately:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

The values can be specified either in pixels (px) or percentage (% of the element’s width).

Margin

Margin generates the space around the outside edge of the box (including borders).
There are properties to set margin either for all sides at once or separately:

  • margin
  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Width and Height

By default browsers set the maximum width for each box. Use width property of the box in order to amend the width of the element.

NOTE when you set the width of an element only content adjusts.
Remember that the full width = width+2*padding+2*border.


Height is set in the same way.
