HTML Introduction

Key principles

HTML describes the structure of a web page. Therefore, HTML elements are the building blocks of the page.

All HTML elements are represented by tags which specify the type of an element.

The template for a tag: < tagName>content</tagName>

a tag name surrounded by angle brackets

Tags normally come in pairs and are called start/opening tag and end/closing tag. They are written in the same way but there is a forward slash inserted before the tag name in the closing tag </tagName>. There is a variety of tags to label different pieces of web content such as “paragraph”(<p>content</p>) or “heading”(<h1>content</h1>).

NOTE Browsers do not display tags BUT use them to determine how to display the content on the page!

The skeletal structure for any web page

There are 3 basic parts of each HTML document which define a structure of a web page:

  • Document type declaration that helps browsers to display the web page correctly. This always comes first and appears only once. It is an instruction to the web browser about the version of HTML used to write the page. Labeled as <!DOCTYPE>. Common DOCTYPE declarations can be found here (depend on the version of HTML used to write the page).
  • Head contains information about the document. It always comes after the document type declaration and in a single copy. A declarative header section does not have any content but must include character set info and the title of the page (<title></title>). It can also include declarations and references for metadata (<meta>), scripts (<script></script>) and style sheets (<link> or <style></style>). However, it is optional.
    <head></head>
  • Body is the actual page content which will be displayed on the web page. It appears once after Head.
    <body></body>

NOTE:

Head and Body blocks are contained within <html></html> tags!

Only the Body section is visible in the browser!
