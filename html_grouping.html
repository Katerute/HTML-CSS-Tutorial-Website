Structural grouping

Group related parts of a document together to create style and interactivity provided by CSS and JavaScript.

How to group elements?

  • Use <div> and <span> tags to enclose related content at document level (as a container for other HTML elements).
    Content of these containers can be treated as a single element.
  • Use class and id attributes to identify a group of elements.

The difference between <div> and <span> tags

<div>

  • defines a block-level section
  • works as a container for other HTML elements

<span>

  • defines an inline fragment of content
  • works as a container for some text

class and id attributes

class

  • <div> and <span> tags can be grouped by adding the class attribute.
  • Containers with the same value of class are associated together.
  • class value can be reused within the document
  • A tag can be in more than one class.

id

  • Any tag can be uniquely identified by id assigned
  • A tag can have only one id attribute (because it is unique)
  • The value of id cannot be reused within the document.
